# 📜 Историческое мобильное приложение

Flutter-приложение для изучения истории. Сверстано по дизайну из Figma Make.

## 🚀 Как запустить

```bash
# 1. Зайди в папку
cd history_app

# 2. Установи зависимости
flutter pub get

# 3. Запусти
flutter run
```

## 📱 Экраны

| Экран | Описание |
|-------|----------|
| **Главная** | Дашборд с прогрессом, стриком и фактом дня |
| **Учиться** | Список исторических эпох с прогрессом |
| **Уроки** | Детальный урок с фактами и контентом |
| **Квиз** | 5 вопросов с объяснениями |
| **Профиль** | Статистика, достижения, настройки |

## 🎨 Дизайн

- **Цвета**: Тёмно-синяя тема с золотым акцентом (#E8A838)
- **Шрифты**: Playfair Display (заголовки) + Lato (текст)
- **Стиль**: Исторический премиум с картами эпох

## 📦 Зависимости

```yaml
google_fonts: ^6.2.1
percent_indicator: ^4.2.3  
flutter_animate: ^4.5.0
```
